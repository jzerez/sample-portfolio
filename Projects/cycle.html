The Cycloid Drawing Machine

Made with lasercut gears and customizable linkage arms, this machine will be able to draw fully customizable cycloids. It is still a work in progress.

Return to Home All Projects

Inspiration

Ever since I learned how to plot in the polar coordinate system on my TI-84, it always was entertaining to plug in random sequences of trigonometric and algebraic functions and see a beautiful, spiral-y graph as an output. The goal of the project is not to understand the trigonometry and algebra behind those beautiful shapes, but to visuallize them geometrically. Also, those graphs are just so pretty! This project is not yet completed, so check back later for future updates!
